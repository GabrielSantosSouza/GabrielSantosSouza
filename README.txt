PASSOS PARA USAR O SISTEMA ART METAL - ORDEM DE SERVIÇO

1. Edite o arquivo:
   config/database.config
   ➜ Exemplo:
     artmetal = 192.168.0.10:C:\banco\artmetal.fdb

2. Edite config.json se quiser mudar usuário/senha do banco.

3. Para compilar:
   - Instale Node.js: https://nodejs.org
   - Execute os comandos:
     npm install
     npm run build

O executável será criado em /dist
